
# Eid Interactive Cube

A simple interactive 3D cube built with **React** and **Three.js** that celebrates Eid!  

When you open the cube’s lid, a festive **“Eid Mubarak”** message appears along with a **confetti effect**.

---

## Features

- 3D textured cube  
- Lid opening animation  
- Confetti effect on reveal  
- “Eid Mubarak” greeting  

---

## Tech Stack

- React  
- Three.js  
- JavaScript / CSS
